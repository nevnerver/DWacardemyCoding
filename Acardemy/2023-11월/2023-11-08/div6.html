텍스트 태그와 css

                    < b > 글씨 굻게 < /b >
                    < strong > 글씨굵게(문장에서 강조) < /strong >
                    < h1~6 > 글의 제목 < /h1 >
                    < p > 문단의 표현 < /p >
                    < br > 텍스트 줄바꿈
                    < blockquote > 텍스트의 인용문 (안쪽 들여쓰기) < /blockquote >
                    < q > 텍스트의인용문(줄바꿈없음) < /q >
                    < pre > < /pre > 
텍스트에 서식지정(작성한 내용 그대로 표현)

                    < figure > < /figure > 
                    < hr > 장면전환, 구획 내 주제변경, 문단분리

                    < i > < /i > 이탤릭체, 기울임
                    < em > < /em > 텍스트의 강조, 중첩사용 가능
                    < mark > < /mark > 텍스트 중요표시, 하이라이트 표시
                    < small > < &/small > 덧붙이는 글, 저작권과 법률 같은거
                    < sup > < /sup > 위 첨자
                    < sub > < /sub > 아래 첨자
                    < del > < /del > 제거텍스트
                    < ins > < /ins > 추가된텍스트
                    < u > < /u > 텍스트에 밑줄
                    < kdb > < /kdb > Ctrl+R
                
                    font-size:20px; 글씨 크기를 20px로 설정
                    font-weight:700; 글씨 굵기 설정, 100단위, 클수록 굵어짐
                    font-family:'궁서'; 글꼴 설정
                    color:red; 글씨 색상
                    text-decoration:underline; 글씨장식 설정
                    text-transform:lowercase; 대소문자 변환
                    text-align:center; 텍스트정렬(수평정렬)
                    letter-spacing:-2px; 글자   간격
                    line-height:30px; 행 높이

                        < /tr > 
                        < tr >
                            < td > < /td >

                    테이블 태그, 줄과 행 표현, 줄-tr, 행-td

                    < div > < /div > 
한 블럭을 표현, 지정한 크기만큼 영역표시가능

                    < fieldset > < /fieldset > 
그룹지정

                    < header > < /header > 
메뉴, 로고, 머리부분

                    < main > < /main > 
본문 내용

                    < footer > < /footer > 
바닥글, 페이지 정보 등

                    < nav > < /nav > 
메뉴 사용

                    < section > < /section > 
본문의 글 주제별로 사용

                    < span > < /span > div보다 작은 영역(줄바꿈 없음)
display - 공간 차지속성( block, inline, inline-block, flex, gird ) position - 태그의 위치 지정(top, right, bottom, left) width - 태그의 너비 height - 태그의 높이 margin - 태그의 바깥쪽 여백 padding - 태그의 안쪽 여백 background - 태그의 배경 색, 이미지 지정 border - 태그의 테두리선 box-shadow - 태그의 그림자
